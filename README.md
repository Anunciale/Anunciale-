# AnunciaLE Cuba Project

## Project Structure

This project is organized into the following main directories:

- **frontend/**: Contains the frontend code and assets.
- **backend/**: Contains the backend code and services.
- **docs/**: Contains project documentation.
- **design/**: Contains design files and prototypes.
- **scripts/**: Contains scripts for automation and tools.

Each directory serves a specific purpose to facilitate collaboration and organization within the project.
